# debug:views
अनुप्रयोगासाठी वर्तमान दृश्ये प्रदर्शित करा.

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | दृश्य टॅग
--status | स्थिती पहा (सक्षम | अक्षम

## Available arguments
Argument | Details
---------|-------------
view-id | दृश्य आयडी

## Examples
* List of views on the site
```
drupal debug:views
```
