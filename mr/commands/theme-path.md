# theme:path
थीमवरील सापेक्ष पथ (किंवा निरपेक्ष पथ) मिळवते.

**Usage:**
```
drupal theme:path [arguments] [options]
thp
```

## Available options
Option | Details
-------|-------------
--absolute | थीम नि:शुल्क पथ परत

## Available arguments
Argument | Details
---------|-------------
theme | Theme name

## Examples
* Get the path of mytheme
```
drupal theme:path  mytheme
```
