# user:password:hash
साधा मजकूर पासवर्ड पासून एक हॅश उत्पन्न करा.

**Usage:**
```
drupal user:password:hash [arguments]
uph
usph
```

## Available arguments
Argument | Details
---------|-------------
password | मजकूर स्वरूपात संकेतशब्द.

## Examples
* Get hash of the word "p455w0rd"
```
drupal user:password:hash  p455w0rd
```
