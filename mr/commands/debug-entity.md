# debug:entity
सिस्टममध्ये उपलब्ध असलेल्या डीबग घटक.

**Usage:**
```
drupal debug:entity [arguments]
de
```

## Available arguments
Argument | Details
---------|-------------
entity-type | commands.debug.entity.arguments.entity-type

## Examples
* Displays the available entities
```
drupal debug:entity
```
