# entity:delete
एखादी विशिष्ट संस्था हटवा.

**Usage:**
```
drupal entity:delete [arguments]
ed
```

## Available arguments
Argument | Details
---------|-------------
entity-definition-id | अस्तित्व परिभाषा आयडी.
entity-id | अस्तित्व आयडी हटविली जाईल.

## Examples
* Delete entity type content using node id
```
drupal entity:delete node 1
```
