# init
कॉन्फिगरेशन फायलींना वापरकर्ता होम डिरेक्ट्रीमध्ये कॉपी करा.

**Usage:**
```
drupal init [options]
```

## Available options
Option | Details
-------|-------------
--destination | फायली कॉपी करण्यासाठी गंतव्य निर्देशिका.
--override | कॉन्फिगरेशन फायली अधिलिखित करा.
--autocomplete | स्वयंपूर्ण साधन फायली ध्वज.
