# debug:permission
सर्व वापरकर्ता परवानग्या प्रदर्शित करते.

**Usage:**
```
drupal debug:permission [arguments]
dp
```

## Available arguments
Argument | Details
---------|-------------
role | वापरकर्ता भूमिका

## Examples
* Displays all the permissions availables on the site
```
drupal debug:permission
```
