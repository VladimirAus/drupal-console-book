# debug:views:plugins
अनुप्रयोगासाठी वर्तमान दृश्ये प्लगइन प्रदर्शित करा.

**Usage:**
```
drupal debug:views:plugins [arguments]
dvp
```

## Available arguments
Argument | Details
---------|-------------
type | दृश्ये प्लगइन प्रकारानुसार फिल्टर करा.

## Examples
* List of views plugins
```
drupal debug:views:plugins
```
