# debug:theme
अनुप्रयोगासाठी वर्तमान थीम प्रदर्शित करते.

**Usage:**
```
drupal debug:theme [arguments]
dt
```

## Available arguments
Argument | Details
---------|-------------
theme | Specific theme to debug.

## Examples
* List of themes on the site
```
drupal debug:theme
```
* Bartik theme information
```
drupal debug:theme bartik
```
