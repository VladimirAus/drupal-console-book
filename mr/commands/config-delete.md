# config:delete
संरचना हटवा

**Usage:**
```
drupal config:delete [arguments]
cd
```

## Available arguments
Argument | Details
---------|-------------
type | कॉन्फिगरेशन प्रकार.
name | संरचने नाव.

## Examples
* एक कॉन्फिगेशन प्रकार आणि कॉन्फिगरेशन नाव प्रदान करा.
```
Drupal config: सर्व सक्रिय हटवा.
```
