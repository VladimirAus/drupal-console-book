# update:execute
विभागाची एक विशिष्ट सुधारणा N कार्य चालवा, किंवा सर्व कार्ये चालवा.

**Usage:**
```
drupal update:execute [arguments]
upex
upe
```

## Available arguments
Argument | Details
---------|-------------
module | मॉड्यूलचे नाव.
update-n | विशिष्ट सुधारणा कार्याची अंमलबजावणी करण्यासाठी

## Examples
* Update all entities
```
drupal update:execute
```
