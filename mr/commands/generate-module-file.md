# generate:module:file
.module फाइल उत्पन्न करा

**Usage:**
```
drupal generate:module:file [options]
gmf
```

## Available options
Option | Details
-------|-------------
--module | मॉड्यूलचे नाव.

## Examples
* Generate the .module file specifying the module name
```
drupal generate:module:file  \
  --module="modulename"
```
