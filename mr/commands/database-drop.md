# database:drop
दिलेल्या डेटाबेसमध्ये सर्व सारण्या ड्रॉप करा.

**Usage:**
```
drupal database:drop [arguments]
dbd
```

## Available arguments
Argument | Details
---------|-------------
database | Settings.php पासून डेटाबेस की.

## Examples
* वितर्क वर निर्दिष्ट डेटाबेसवर सारण्या ड्रॉप करा.
```
drupal database:drop \
  <database>
```
