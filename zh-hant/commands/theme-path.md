# theme:path
取得版型所在相關或絕對路徑

**Usage:**
```
drupal theme:path [arguments] [options]
thp
```

## Available options
Option | Details
-------|-------------
--absolute | 回傳絕對路徑

## Available arguments
Argument | Details
---------|-------------
theme | Theme name

## Examples
* Get the path of mytheme
```
drupal theme:path  mytheme
```
