# debug:rest
列出網站應用中的 REST 資源

**Usage:**
```
drupal debug:rest [arguments] [options]
rede
```

## Available options
Option | Details
-------|-------------
--authorization | 指定 REST 資源狀態：enabled | disabled

## Available arguments
Argument | Details
---------|-------------
resource-id | REST ID

## Examples
* Displays rest hooks
```
drupal debug:rest
```
