# list
사용할 수 있는 모든 명령어의 목록을 조회합니다.

**Usage:**
```
drupal list [arguments] [options]
```

## Available options
Option | Details
-------|-------------
--xml | 목록을 XML로 출력
--raw | 명령어 목록을 그대로 출력
--format | 출력 형식 (txt, xml, json, md)

## Available arguments
Argument | Details
---------|-------------
namespace | 네임스페이스명
