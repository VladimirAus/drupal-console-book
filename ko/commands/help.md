# help
명령어에 대한 도움말을 보여줍니다.

**Usage:**
```
drupal help [arguments] [options]
```

## Available options
Option | Details
-------|-------------
--xml | 목록을 XML로 출력
--raw | 명령어 목록을 그대로 출력
--format | 출력 형식 (txt, xml, json, md)

## Available arguments
Argument | Details
---------|-------------
command_name | The command name
