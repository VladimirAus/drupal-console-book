# user:password:hash
Сгенерировать хеш для пароля.

**Usage:**
```
drupal user:password:hash [arguments]
uph
usph
```

## Available arguments
Argument | Details
---------|-------------
password | Пароль(и) в текстовом формате

## Examples
* Get hash of the word "p455w0rd"
```
drupal user:password:hash  p455w0rd
```
