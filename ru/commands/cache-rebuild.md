# cache:rebuild
Перестроить и очистить весь кеш сайта.

**Usage:**
```
drupal cache:rebuild [arguments]
cr
```

## Available arguments
Argument | Details
---------|-------------
cache | Очистить только конкретный кеш.

## Examples
* Перестроить все кеши
```
drupal cr all
```
* Перестроить кеш отображения
```
drupal cr discovery
```
