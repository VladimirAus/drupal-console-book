# update:execute
Выполнить конкретный update_N хук в модуле или выполнить все

**Usage:**
```
drupal update:execute [arguments]
upex
upe
```

## Available arguments
Argument | Details
---------|-------------
module | Имя модуля.
update-n | Конкретный update_N хук будет выполнен

## Examples
* Update all entities
```
drupal update:execute
```
