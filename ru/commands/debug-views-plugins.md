# debug:views:plugins
Отображает существующие плагины Представлений для приложения

**Usage:**
```
drupal debug:views:plugins [arguments]
dvp
```

## Available arguments
Argument | Details
---------|-------------
type | Отфильтровать плагины Представлений по типу

## Examples
* List of views plugins
```
drupal debug:views:plugins
```
