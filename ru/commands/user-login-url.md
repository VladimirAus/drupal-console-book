# user:login:url
Возвращает одноразовый url для входа.

**Usage:**
```
drupal user:login:url [arguments]
ulu
uslu
```

## Available arguments
Argument | Details
---------|-------------
user-id | Идентификатор пользователя.

## Examples
* Get one time login url for user id 1
```
drupal user:login:url  1
```
