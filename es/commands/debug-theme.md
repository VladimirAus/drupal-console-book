# debug:theme
Despliega los temas actuales para la aplicación

**Usage:**
```
drupal debug:theme [arguments]
dt
```

## Available arguments
Argument | Details
---------|-------------
theme | Tema específico a depurar

## Examples
* List of themes on the site
```
drupal debug:theme
```
* Bartik theme information
```
drupal debug:theme bartik
```
