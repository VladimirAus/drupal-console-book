# user:password:hash
Crea un hash a partir de una contraseña en texto plano.

**Usage:**
```
drupal user:password:hash [arguments]
uph
usph
```

## Available arguments
Argument | Details
---------|-------------
password | Contraseña(s) en formato texto

## Examples
* Get hash of the word "p455w0rd"
```
drupal user:password:hash  p455w0rd
```
