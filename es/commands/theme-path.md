# theme:path
Devuelve la ruta relativa al theme (o ruta absoluta)

**Usage:**
```
drupal theme:path [arguments] [options]
thp
```

## Available options
Option | Details
-------|-------------
--absolute | Devuelve la ruta absoluta del theme

## Available arguments
Argument | Details
---------|-------------
theme | Theme name

## Examples
* Get the path of mytheme
```
drupal theme:path  mytheme
```
