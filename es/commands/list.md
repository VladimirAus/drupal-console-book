# list
Enlista comandos

**Usage:**
```
drupal list [arguments] [options]
```

## Available options
Option | Details
-------|-------------
--xml | Para desplegar la lista como XML
--raw | Para desplegar un lista completa de comandos
--format | Formato de salida (txt, xml, json, or md)

## Available arguments
Argument | Details
---------|-------------
namespace | Nombre del espacio de nombres
