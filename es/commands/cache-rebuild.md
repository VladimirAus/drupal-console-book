# cache:rebuild
Reconstruye y limpia todas las cachés del sitio.

**Usage:**
```
drupal cache:rebuild [arguments]
cr
```

## Available arguments
Argument | Details
---------|-------------
cache | Reconstruir sólo una cache determinada.

## Examples
* Reconstruir todas las cachés
```
drupal cr all
```
* Reconstruir la caché "discovery"
```
drupal cr discovery
```
