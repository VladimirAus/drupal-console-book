# container:debug
Muestra los servicios actuales de la aplicación.

**Uso:**
```
$ drupal container:debug [arguments] [options]
```

## Opciones disponibles
Opción | Detalles
-------|-------------
--parameters | Nombre del servicio.

## Argumentos disponibles
Argumento | Detalles
---------|-------------
service | Nombre del servicio.
