# site:mode
Schimbă configurarile de performanţă ale sistemului

**Usage:**
```
drupal site:mode [arguments]
smo
```

## Available arguments
Argument | Details
---------|-------------
environment | Numele mediului

## Examples
* Switch system to prod
```
drupal site:mode  prod
```
* Switch system to dev
```
drupal site:mode  dev
```
