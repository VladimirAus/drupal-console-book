# debug:rest
Afişează resursele REST curente.

**Usage:**
```
drupal debug:rest [arguments] [options]
rede
```

## Available options
Option | Details
-------|-------------
--authorization | Starea resursei REST activat | dezactivat

## Available arguments
Argument | Details
---------|-------------
resource-id | ID-ul resursei

## Examples
* Displays rest hooks
```
drupal debug:rest
```
