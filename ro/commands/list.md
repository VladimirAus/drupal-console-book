# list
Listele de comenzi

**Usage:**
```
drupal list [arguments] [options]
```

## Available options
Option | Details
-------|-------------
--xml | Afișarea listei ca XML
--raw | Afișarea brută a listei de comenzi
--format | Formatul de afișare (txt, xml, json, sau md)

## Available arguments
Argument | Details
---------|-------------
namespace | Numele spațiului de lucru
