# rest:debug
Exibir o recurso REST atual para a aplicação

**Uso:**
```
$ drupal rest:debug [arguments] [options]
$ rede  
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--authorization | Recurso REST habilitado | desabilitado

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
resource-id | REST ID
