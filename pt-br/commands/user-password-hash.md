# user:password:hash
Gerar o hash de uma senha em formato texto.

**Usage:**
```
drupal user:password:hash [arguments]
uph
usph
```

## Available arguments
Argument | Details
---------|-------------
password | Senha(s) em formato texto

## Examples
* Get hash of the word "p455w0rd"
```
drupal user:password:hash  p455w0rd
```
