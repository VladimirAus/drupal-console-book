# debug:test
Listar testes unitários disponíveis para a aplicação.

**Usage:**
```
drupal debug:test [arguments] [options]
td
```

## Available options
Option | Details
-------|-------------
--test-class | Classe do teste

## Available arguments
Argument | Details
---------|-------------
group | Grupo

## Examples
* 
```
drupal debug:test
```
