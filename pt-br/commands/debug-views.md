# debug:views
Exibir informações sobre a View atual

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | Tag da View
--status | Status da View (Habilitado|Desabilitado)

## Available arguments
Argument | Details
---------|-------------
view-id | ID da View

## Examples
* List of views on the site
```
drupal debug:views
```
