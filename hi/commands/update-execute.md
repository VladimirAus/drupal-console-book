# update:execute
मोड्यूल के किसी विशेष नवीनीकरण N फंक्शन को चलायें या सभी को चलायें।

**Usage:**
```
drupal update:execute [arguments]
upex
upe
```

## Available arguments
Argument | Details
---------|-------------
module | मोड्यूल का नाम।
update-n | विशिष्ट अपडेट N फंक्शन एक्सेक्युट किया जाना

## Examples
* Update all entities
```
drupal update:execute
```
