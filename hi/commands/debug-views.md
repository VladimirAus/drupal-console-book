# debug:views
अनुप्रयोग के वर्तमान व्यूज साधनो को दिखाएँ

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | व्यू टैग
--status | व्यू स्टेटस (Enabled|Disabled)

## Available arguments
Argument | Details
---------|-------------
view-id | व्यू आईडी

## Examples
* List of views on the site
```
drupal debug:views
```
