# site:mode
प्रणाली की कार्य-निष्पादन व्यवस्था को बदलें

**Usage:**
```
drupal site:mode [arguments]
smo
```

## Available arguments
Argument | Details
---------|-------------
environment | एनवायरनमेंट नाम (dev|prod)

## Examples
* Switch system to prod
```
drupal site:mode  prod
```
* Switch system to dev
```
drupal site:mode  dev
```
