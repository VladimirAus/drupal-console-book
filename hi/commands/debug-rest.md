# debug:rest
आवेदन के लिए मौजूदा REST संसाधन को प्रदर्शित करें

**Usage:**
```
drupal debug:rest [arguments] [options]
rede
```

## Available options
Option | Details
-------|-------------
--authorization | REST साधन की स्थिति (चालू है । बंद है )

## Available arguments
Argument | Details
---------|-------------
resource-id | Rest क्रमांक

## Examples
* Displays rest hooks
```
drupal debug:rest
```
