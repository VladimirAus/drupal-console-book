# user:password:hash
सरल पासवर्ड से हैश उत्पन्न करें।

**Usage:**
```
drupal user:password:hash [arguments]
uph
usph
```

## Available arguments
Argument | Details
---------|-------------
password | पासवर्ड(स) टेक्स्ट फॉर्मेट में

## Examples
* Get hash of the word "p455w0rd"
```
drupal user:password:hash  p455w0rd
```
