# user:login:url
एक बार काम करने वाली उपभोग्ता लोगिन URL देता है।

**Usage:**
```
drupal user:login:url [arguments]
ulu
uslu
```

## Available arguments
Argument | Details
---------|-------------
user-id | यूजर आईड।

## Examples
* Get one time login url for user id 1
```
drupal user:login:url  1
```
