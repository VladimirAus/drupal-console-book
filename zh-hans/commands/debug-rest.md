# debug:rest
显示当前 REST 资源

**Usage:**
```
drupal debug:rest [arguments] [options]
rede
```

## Available options
Option | Details
-------|-------------
--authorization | REST 资源状态 启用 | 禁用

## Available arguments
Argument | Details
---------|-------------
resource-id | REST 资源 ID

## Examples
* Displays rest hooks
```
drupal debug:rest
```
