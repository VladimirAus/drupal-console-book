# debug:state
显示当前状态的键(s)

**Usage:**
```
drupal debug:state [arguments]
dst
```

## Available arguments
Argument | Details
---------|-------------
key | 状态的键

## Examples
* List of the states on the site
```
drupal debug:state
```
* Displays a detail of the state install_task tok from the list of states
```
drupal debug:state install_task
```
