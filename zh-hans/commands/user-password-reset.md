# user:password:reset
为指定用户重设密码.

**Usage:**
```
drupal user:password:reset [arguments]
upr
uspr
```

## Available arguments
Argument | Details
---------|-------------
user | 用户ID
password | 文本格式的密码

## Examples
* Update password specifying the user id and the new password
```
drupal user:password:reset  2 p455w0rd
```
