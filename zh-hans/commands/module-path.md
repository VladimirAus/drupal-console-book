# module:path
返回模块的相对路径(或绝对路径)

**Usage:**
```
drupal module:path [arguments] [options]
mop
```

## Available options
Option | Details
-------|-------------
--absolute | 返回模块绝对路径

## Available arguments
Argument | Details
---------|-------------
module | 模块名称 (机读名称)

## Examples
* Get the relative path of the module specifying the module name
```
drupal module:path  modulename
```
