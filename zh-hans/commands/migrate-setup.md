# migrate:setup
导入或创建数据库迁移

**Usage:**
```
drupal migrate:setup [options]
mis
```

## Available options
Option | Details
-------|-------------
--db-type | Drupal 数据库类型
--db-host | 数据库主机
--db-name | 数据库名称
--db-user | 数据库用户
--db-pass | 数据库密码
--db-prefix | 数据库前缀
--db-port | 数据库端口
--source-base_path | commands.migrate.setup.options.source-base-path
