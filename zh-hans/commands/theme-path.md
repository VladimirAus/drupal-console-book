# theme:path
返回主题的相对路径 (或绝对路径)

**Usage:**
```
drupal theme:path [arguments] [options]
thp
```

## Available options
Option | Details
-------|-------------
--absolute | 返回主题绝对路径

## Available arguments
Argument | Details
---------|-------------
theme | Theme name

## Examples
* Get the path of mytheme
```
drupal theme:path  mytheme
```
