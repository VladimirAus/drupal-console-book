# debug:event
显示当前事件

**Usage:**
```
drupal debug:event [arguments]
dev
```

## Available arguments
Argument | Details
---------|-------------
event | 调试的事件

## Examples
* List all the events that could be debugged
```
drupal debug:evet
```
* Show the information for the kernel.request event
```
drupal debug:event kernel.request
```
