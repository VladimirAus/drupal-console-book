# theme:install
安装主题

**Usage:**
```
drupal theme:install [arguments] [options]
thi
ti
```

## Available options
Option | Details
-------|-------------
--set-default | 设置主题为默认主题

## Available arguments
Argument | Details
---------|-------------
theme | commands.theme.install.options.module

## Examples
* Install theme specifying the name
```
drupal theme:install  mytheme
```
