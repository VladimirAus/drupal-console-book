# help
显示帮助

**Usage:**
```
drupal help [arguments] [options]
```

## Available options
Option | Details
-------|-------------
--xml | 以 XML 格式输出
--raw | 以 raw 格式输出
--format | 输出格式（txt, xml, json, or md）

## Available arguments
Argument | Details
---------|-------------
command_name | The command name
