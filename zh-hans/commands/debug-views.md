# debug:views
显示网站当前视图资源

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | 视图标签
--status | 视图状态(已启用|已禁用)

## Available arguments
Argument | Details
---------|-------------
view-id | 视图ID

## Examples
* List of views on the site
```
drupal debug:views
```
