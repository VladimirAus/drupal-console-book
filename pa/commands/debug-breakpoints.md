# debug:breakpoints
ਵੇਖਾਉਦਾ ਹੈ ਬਰੇਕ ਪੁਆਇੰਟ ਐਪਲੀਕੇਸ਼ਨ ਵਿੱਚ ਉਪਲੱਬਧ

**Usage:**
```
drupal debug:breakpoints [arguments]
dbre
```

## Available arguments
Argument | Details
---------|-------------
group | ਬਰੇਕ ਪੁਆਇੰਟ ਗਰੁੱਪ ਦਾ ਨਾਮ

## Examples
* Provide a group name.
```
drupal breakpoints:debug bartik
```
