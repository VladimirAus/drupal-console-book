# user:login:url
Retourne une url de connection à validité unique pour un utilisateur.

**Usage:**
```
drupal user:login:url [arguments]
ulu
uslu
```

## Available arguments
Argument | Details
---------|-------------
user-id | Identifiant de l'utilisateur

## Examples
* Get one time login url for user id 1
```
drupal user:login:url  1
```
