# module:path
モジュールの相対パスまたは絶対パスを返す

**Usage:**
```
drupal module:path [arguments] [options]
mop
```

## Available options
Option | Details
-------|-------------
--absolute | 絶対パスを返す

## Available arguments
Argument | Details
---------|-------------
module | モジュール名 (システム内部名称)

## Examples
* Get the relative path of the module specifying the module name
```
drupal module:path  modulename
```
