# debug:views
Viewsのリソースを表示する

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | ビューのタグ
--status | ビューのステータス (Enabled|Disabled)

## Available arguments
Argument | Details
---------|-------------
view-id | ビューのID

## Examples
* List of views on the site
```
drupal debug:views
```
