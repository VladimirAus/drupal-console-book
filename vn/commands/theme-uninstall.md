# theme:uninstall
Gỡ cài đặt theme hay các themes trong ứng dụng

**Usage:**
```
drupal theme:uninstall [arguments]
thu
tu
```

## Available arguments
Argument | Details
---------|-------------
theme | commands.theme.uninstall.options.module

## Examples
* Uninstall theme specifying the name
```
drupal theme:uninstall  mytheme
```
