# debug:views
Hiển thị các views resources hiện tại cho ứng dụng

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | View tag
--status | Trạng thái View (Bật|Tắt)

## Available arguments
Argument | Details
---------|-------------
view-id | View ID

## Examples
* List of views on the site
```
drupal debug:views
```
