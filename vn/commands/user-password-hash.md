# user:password:hash
Tạo một hash từ một mật khẩu thô

**Usage:**
```
drupal user:password:hash [arguments]
uph
usph
```

## Available arguments
Argument | Details
---------|-------------
password | Mật khẩu có định dạng text

## Examples
* Get hash of the word "p455w0rd"
```
drupal user:password:hash  p455w0rd
```
