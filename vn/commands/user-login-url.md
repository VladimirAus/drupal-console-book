# user:login:url
Quay lại một url đăng nhập của người dùng một lần.

**Usage:**
```
drupal user:login:url [arguments]
ulu
uslu
```

## Available arguments
Argument | Details
---------|-------------
user-id | ID người dùng.

## Examples
* Get one time login url for user id 1
```
drupal user:login:url  1
```
