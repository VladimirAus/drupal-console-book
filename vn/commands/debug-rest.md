# debug:rest
Hiển thị rest resource hiện có cho ứng dụng

**Usage:**
```
drupal debug:rest [arguments] [options]
rede
```

## Available options
Option | Details
-------|-------------
--authorization |  Trạng thái rest resource bật | tắt

## Available arguments
Argument | Details
---------|-------------
resource-id | Rest ID

## Examples
* Displays rest hooks
```
drupal debug:rest
```
