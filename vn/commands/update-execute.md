# update:execute
Tiến hành một hàm cụ thể cập nhật N trong một module, hoặc tiến hành tất cả

**Usage:**
```
drupal update:execute [arguments]
upex
upe
```

## Available arguments
Argument | Details
---------|-------------
module | Tên module.
update-n | Hàm cập nhật N cụ thể để được tiến hành

## Examples
* Update all entities
```
drupal update:execute
```
