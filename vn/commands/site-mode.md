# site:mode
Chuyển đổi cấu hình hiệu năng của hệ thống

**Usage:**
```
drupal site:mode [arguments]
smo
```

## Available arguments
Argument | Details
---------|-------------
environment | Tên môi trường (dev|prod)

## Examples
* Switch system to prod
```
drupal site:mode  prod
```
* Switch system to dev
```
drupal site:mode  dev
```
