# debug:test
Danh sách các đơn vị thử nghiệm có sẵn của ứng dụng.

**Usage:**
```
drupal debug:test [arguments] [options]
td
```

## Available options
Option | Details
-------|-------------
--test-class | Lớp thử nghiệm

## Available arguments
Argument | Details
---------|-------------
group | Nhóm

## Examples
* 
```
drupal debug:test
```
