# user:login:url
Egyszer használatos felhasználói bejelentkezési URL-címet ad vissza.

**Usage:**
```
drupal user:login:url [arguments]
ulu
uslu
```

## Available arguments
Argument | Details
---------|-------------
user-id | Felhasználói azonosító.

## Examples
* Get one time login url for user id 1
```
drupal user:login:url  1
```
