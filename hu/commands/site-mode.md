# site:mode
Rendszerteljesítmény beállításának átváltása

**Usage:**
```
drupal site:mode [arguments]
smo
```

## Available arguments
Argument | Details
---------|-------------
environment | Környezet neve [dev, prod]

## Examples
* Switch system to prod
```
drupal site:mode  prod
```
* Switch system to dev
```
drupal site:mode  dev
```
