# debug:views
Alkalmazás aktuális views erőforrásainak megjelenítése

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | Nézet címkéje
--status | Nézet állapota (engedélyezett|tiltott)

## Available arguments
Argument | Details
---------|-------------
view-id | Nézet azonosítója

## Examples
* List of views on the site
```
drupal debug:views
```
