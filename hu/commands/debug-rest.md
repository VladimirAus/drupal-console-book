# debug:rest
Aktuális REST-erőforrás megjelenítése

**Usage:**
```
drupal debug:rest [arguments] [options]
rede
```

## Available options
Option | Details
-------|-------------
--authorization | Engedélyezett | tilott állapotú REST-erőforrás

## Available arguments
Argument | Details
---------|-------------
resource-id | REST-azonosító

## Examples
* Displays rest hooks
```
drupal debug:rest
```
