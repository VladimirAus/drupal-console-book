# cache:rebuild
Bangun ulang dan bersihkan semua cache situs.

**Usage:**
```
drupal cache:rebuild [arguments]
cr
```

## Available arguments
Argument | Details
---------|-------------
cache | Hanya hapus cache tertentu.

## Examples
* Bangun ulang semua cache
```
drupal cr all
```
* Bangun ulang cache discovery
```
drupal cr discovery
```
